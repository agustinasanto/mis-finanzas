Setup inicial
  1. Creá un proyecto en Supabase.
  2. En SQL Editor corré el script sugerido para crear la tabla.
  3. Copiá tu Project URL y la anon/public key.
  4. Elegí un código personal para separar tus datos.
create table if not exists public.mf_states ( code text primary key, payload jsonb not null, updated_at timestamptz not null default now() ); alter table public.mf_states enable row level security; create policy "mf_states_public_rw" on public.mf_states for all using (true) with check (true); alter publication supabase_realtime add table public.mf_states;
Vamos a validar URL y key, leer tu fila por código y hacer una escritura de prueba segura antes de guardar.
